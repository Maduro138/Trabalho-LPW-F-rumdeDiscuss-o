Aprenda a tomar decisões no seu programa

O que são estruturas condicionais?

Estruturas condicionais permitem que o programa escolha caminhos diferentes conforme uma condição (verdadeira/falsa). As principais em C são if, else if, else e switch.

if / else

Use if para executar código quando a condição for verdadeira; else para o caso contrário.

#include <stdio.h>

int main() {
    int idade;
    printf("Digite sua idade: ");
    scanf("%d", &idade);

    if (idade >= 18) {
        printf("Você é maior de idade.\n");
    } else {
        printf("Você é menor de idade.\n");
    }
    return 0;
}

else if (múltiplas condições)

Para verificar várias condições em sequência use else if.

#include <stdio.h>

int main() {
    int nota;
    printf("Digite a nota (0-100): ");
    scanf("%d", ¬a);

    if (nota >= 90) {
        printf("A\n");
    } else if (nota >= 80) {
        printf("B\n");
    } else if (nota >= 70) {
        printf("C\n");
    } else {
        printf("Reprovado\n");
    }
    return 0;
}

switch / case

Use switch quando quiser comparar uma variável com vários valores constantes.

#include <stdio.h>

int main() {
    int dia;
    printf("Digite o número do dia (1=Dom,2=Seg,...7=Sab): ");
    scanf("%d", &dia);

    switch (dia) {
      case 1: printf("Domingo\n"); break;
      case 2: printf("Segunda\n"); break;
      case 3: printf("Terça\n"); break;
      case 4: printf("Quarta\n"); break;
      case 5: printf("Quinta\n"); break;
      case 6: printf("Sexta\n"); break;
      case 7: printf("Sábado\n"); break;
      default: printf("Número inválido\n"); break;
    }
    return 0;
}

Boas práticas

  • Coloque chaves {} mesmo para blocos de uma linha — melhora leitura e evita erros.
  • Use break nos case do switch para evitar "fall-through" indesejado.
  • Prefira condições claras e curtas; se necessário, use funções auxiliares.

Exercícios

  1. Leia um número inteiro e informe se ele é positivo, negativo ou zero.
  2. Escreva um programa que leia a idade e informe a categoria: criança (<12), adolescente (12–17), adulto (18–64), idoso (≥65).
  3. Faça um menu com switch que receba uma opção (1–3) e mostre uma mensagem diferente para cada opção; trate opção inválida.
