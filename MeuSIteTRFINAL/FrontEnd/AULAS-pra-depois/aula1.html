Aula 1 — Introdução à Linguagem C

Objetivos: entender o que é C, estrutura básica, compilar/executar e usar printf/scanf.

1. O que é C

C é uma linguagem de programação criada por Dennis Ritchie (1972) nos Bell Labs. É compacta, eficiente e base para muitas outras linguagens (C++, Java, C#). Muito usada em sistemas, embarcados e software de alto desempenho.

  • Características: performance, controle de memória (ponteiros), sintaxe concisa.
  • Onde aparece: sistemas operacionais, drivers, firmware, jogos, bibliotecas.

3. printf e formatação

printf imprime texto com especificadores de formato.


int a = 10;
float b = 3.14;
printf("Inteiro: %d\n", a);
printf("Real: %.2f\n", b);
printf("Caractere: %c\n", 'A');
printf("Texto: %s\n", "Olá");

Principais especificadores: %d (int), %f (float/double), %c (char), %s (string).

4. Entrada: scanf

Use scanf para ler dados do usuário — lembre-se do & para variáveis (exceto arrays).

Exemplo: ler nome e idade
Testar online 
#include <stdio.h>

int main() {
    char nome[50];
    int idade;

    printf("Digite seu nome: ");
    scanf("%49s", nome);       // %s lê até o primeiro espaço
    printf("Digite sua idade: ");
    scanf("%d", &idade);

    printf("Olá %s, você tem %d anos.\n", nome, idade);
    return 0;
}

Observação: scanf("%s", nome) lê até o primeiro espaço; para ler linhas com espaços use fgets (ou funções adicionais).

5. Comentários e erros comuns

Comentários:

// comentário de uma linha
/* comentário
   de várias linhas */

Erros comuns:

  • Esquecer ponto e vírgula ;.
  • Não incluir <stdio.h> antes de usar printf/scanf.
  • Esquecer & em scanf para variáveis escalares.

6. Exercício resolvido

Peça o nome e a idade e imprima: Olá [nome], você tem [idade] anos!

Resumo rápido

O que lembrar

  • Programa começa em int main().
  • Use printf para saída e scanf para entrada.
  • Toda instrução termina com ;.
  • Inclua <stdio.h> para entrada e saída básicas.
