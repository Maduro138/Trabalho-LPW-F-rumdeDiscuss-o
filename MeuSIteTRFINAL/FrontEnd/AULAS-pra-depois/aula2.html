Aula 2 — Variáveis, Tipos e Operadores (C)

Aprendendo a guardar valores e fazer operações

O que é uma variável?

Variáveis são nomes que usamos para guardar valores na memória do computador. Cada variável tem um tipo que diz que tipo de valor ela armazena (inteiro, real, caractere, etc.).

Tipos básicos em C

  • int — números inteiros (ex: -2, 0, 10).
  • float — números com casas decimais (ex: 3.14).
  • double — float com maior precisão.
  • char — um único caractere (ex: 'A').

Declarar e inicializar variáveis

Declaração: ao criar a variável você diz seu tipo e nome. Inicialização: você dá um valor inicial.

int idade;            // declara a variável 'idade' do tipo inteiro
int ano = 2025;       // declara e inicializa
float nota = 8.5;     // número com casas decimais
char letra = 'A';     // caractere

Observação: em C cada instrução termina com ponto e vírgula ;.

 Formatadores e printf

Para mostrar variáveis com printf usamos especificadores:

int a = 5;
float b = 2.5;
char c = 'Z';

printf("a = %d\n", a);      // %d para inteiros
printf("b = %.2f\n", b);    // %f para reais (%.2f = 2 casas)
printf("c = %c\n", c);      // %c para caractere

Operadores

Operadores aritméticos: +, -, *, /, % (resto). Operadores relacionais: ==, !=, <, >, <=, >=. 

int x = 7, y = 3;
int soma = x + y;      // 10
int prod = x * y;      // 21
int divi = x / y;      // 2  (divisão inteira)
int resto = x % y;     // 1  (resto da divisão)

Conversão de tipos (casting)

Para evitar divisão inteira, convertemos para float:

int a = 5, b = 2;
float resultado = (float)a / b;  // 2.5
printf("Resultado = %.2f\n", resultado);

Exercícios

  1. Declare variáveis para armazenar idade, altura e inicial do nome.
  2. Leia dois inteiros e imprima a soma, a diferença e o produto.
  3. Leia dois números inteiros e mostre sua divisão com duas casas decimais (use casting).
  4. Explique com palavras a diferença entre float e double.
