Aula 4 — Operadores em C

Aprenda a usar operadores aritméticos, relacionais e lógicos.

1. Operadores Aritméticos

Aritméticos realizam contas básicas.

  • + Soma
  • - Subtração
  • * Multiplicação
  • / Divisão
  • % Resto da divisão inteira (módulo)
#include <stdio.h>

int main() {
    int a = 10, b = 3;
    printf("Soma: %d\n", a + b);
    printf("Subtração: %d\n", a - b);
    printf("Multiplicação: %d\n", a * b);
    printf("Divisão: %d\n", a / b);
    printf("Resto: %d\n", a % b);
    return 0;
}

2. Operadores Relacionais

Usados para comparar valores. Retornam 1 (verdadeiro) ou 0 (falso).

  • == Igual
  • != Diferente
  • > Maior
  • < Menor
  • >= Maior ou igual
  • <= Menor ou igual
#include <stdio.h>

int main() {
    int x = 5, y = 8;
    printf("%d\n", x > y);
    printf("%d\n", x == 5);
    printf("%d\n", y != 3);
    return 0;
}

3. Operadores Lógicos

Usados para criar condições compostas.

  • && E lógico
  • || OU lógico
  • ! Negação
#include <stdio.h>

int main() {
    int idade = 20;
    int habilitacao = 1;

    if (idade >= 18 && habilitacao == 1) {
        printf("Pode dirigir.\n");
    } else {
        printf("Não pode dirigir.\n");
    }
    return 0;
}

4. Precedência dos Operadores

Ordem em que as operações são executadas:

  • 1º () Parênteses
  • 2º !
  • 3º *, /, %
  • 4º +, -
  • 5º Relacionais
  • 6º &&
  • 7º ||

5. Exercícios

  1. Leia dois números e mostre todas as operações aritméticas entre eles.
  2. Leia a idade de uma pessoa e diga se ela pode votar e/ou dirigir.
  3. Faça um programa que receba um número e diga se ele está entre 10 e 50 usando operadores lógicos.
