O que é C?

C é uma linguagem de programação de propósito geral, criada nos anos 70. É rápida, próxima ao hardware e é a base para muitas outras linguagens.

Por que aprender C?

  • Controle baixo de memória (ponteiros).
  • Base para sistemas operacionais, compiladores e embarcados.
  • Excelente para aprender conceitos fundamentais.

Estrutura mínima de um programa em C

#include <stdio.h>

int main(void) {
    printf("Olá, mundo!\n");
    return 0;
}

Salve em ola.c e compile com gcc ola.c -o ola.

Ponteiros (exemplo básico)

int x = 10;
int *p = &x;
printf("x = %d, *p = %d\n", x, *p);

Exemplo: função que soma dois números

#include <stdio.h>

int soma(int a, int b) {
    return a + b;
}

int main(void) {
    printf("3 + 4 = %d\n", soma(3,4));
    return 0;
